== Description ==

This plugin is an extension for Woocommerce and Client Dash and helps users and webmasters by taking the sales reports from Woocommerce and placing it in a logical location within the Client Dash interface.

== Installation ==

Using this plugin is very simple. All you have to do is:

1. Upload the `client-dash-woocommerce-add-on` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
